Desafío – Layout de E-commerce con CSS Grid

Descripción
Este proyecto corresponde al desarrollo de un layout de e-commerce utilizando **CSS Grid** y **Flexbox**, de acuerdo con los requerimientos establecidos en el desafío.

El objetivo principal es aplicar correctamente las técnicas de maquetación vistas en clases, separando la estructura general del sitio mediante Grid y organizando los componentes internos con Flexbox.

Estructura del Layout
El layout está compuesto por las siguientes secciones:

- **Header**: encabezado principal del sitio.
- **Sidebar**: barra lateral con categorías de productos.
- **Main**: sección principal que contiene la grilla de productos.
- **Footer**: pie de página con información general.

La estructura general se implementa utilizando `grid-template-areas`.

Tecnologías Utilizadas
- HTML5
- CSS
  - CSS Grid
  - Flexbox

Detalles Técnicos

CSS Grid
- Se utiliza Grid para la estructura principal del sitio.
- La sección de productos está organizada en una grilla de **3 columnas**.
- El layout se adapta automáticamente cuando hay menos productos.

Flexbox
- Se utiliza Flexbox en:
  - La barra lateral (categorías).
  - Las tarjetas de producto para alinear imagen, texto y botón.

 Diseño
- Fondo gris claro para el área principal y sidebar.
- Tarjetas de producto con fondo blanco.
- Botón de acción ubicado al final
